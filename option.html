Explanation of use for CSS categories

  1. Linear Gradient: I applied a linear gradient background to the header using the linear-gradient function. Creating a smooth transition between colors in the background of the header.
  2. @font-face Rule: I used two fonts: "Dancing Script" and "Amatic SC", embedded from Google Fonts. Giving the header a custom design. I added the links to Google Fonts in the head section of the home.html page.
  3. Horizontal Scrolling: I implemented horizontal scrolling in the navigation menu that points to specific parts of the site, using flexbox and overflow-x. With flex-wrap: nowrap and overflow-x: auto to allow users to scroll horizontally when necessary.
  4. Box Shadow: I applied a box-shadow to the scroll menu, images, and iframe to create a 3D effect. The subtle shadow enhances visibility, making these elements stand out.
  5. Text Shadow: I applied text-shadow when hovering over the menu items, creating a glowing effect that adds emphasis to the pointed text.
  6. Use of Alpha Transparency: I applied alpha transparency in both the box shadows and text shadows to enhance the visual effects. Additionally, I used alpha with RGBA in the overlay on the main image for copyright attribution, ensuring legibility without completely covering the image.
  7. CSS Structural Pseudo-Classes: I used the :last-child pseudo-class to bold the last paragraph of each article. Additionally, I applied the :nth-child() pseudo-class with a numeric expression (n+3) and (n+2) to make the image layout responsive. Using @media queries, two images are displayed when the viewport is less than 768px wide, and only one image when the viewport is less than 480px wide.
  8. Substring Matching Attribute Selectors: In the footer, I used a substring matching attribute selector [href^="mailto:"] to style the email link. Applying a unique color and font weight to email addresses, making it stand out in the footer.
